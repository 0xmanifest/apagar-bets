## 🧬 O Motor Científico
A simulação realiza 10.000 iterações matemáticas por jogo. As nossas variáveis consideram:
- **Fatores Fisiológicos:** Altitude exata das cidades, Fadiga (Jetlag) e Idade Média.
- **Sentimento Contrariano (NLP):** O algoritmo de linguagem natural mapeia o "Salto Alto" (soberba) ou a "Fritura" (crise) dos times.
- **Efeito Manada (O Ponto Cego):** Se 90% do público apostar de forma emocional e errada, nosso motor identifica o Valor Esperado Positivo (EV+) oculto.
